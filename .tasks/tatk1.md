# 实现一个在线excel编辑器
## 功能如下
- 单元格编辑
- 公式计算
- 样式设置
- 数据校验
- 数据导入导出
- 只是大数据量的数据渲染
- 自行模拟滚动条(详细描述该技术实现原理)


## 技术栈如下
- vue2.7.16
- typescript
- ant-design-vue
- 不引入第三方，尽量使用js元原生逻辑(例如canvas渲染等)
